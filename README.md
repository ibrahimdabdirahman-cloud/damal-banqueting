# Damal Banqueting

Single-page site for **Damal Banqueting** — East London's premier Somali banqueting hall
(Stepney Green). Self-contained static HTML, no build step. Event bookings and reviews are
wired to the shared Masul forms backend via `masul-widget.js`.

## Deploy
Static site — push to GitHub and connect to Vercel (Framework: Other).
